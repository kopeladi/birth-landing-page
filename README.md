# הכנה ללידה שמחה ורגועה | עדולה

דף נחיתה לקורס הכנה ללידה של **עדי קופל לוי — עדולה**.

## על הפרויקט

דף נחיתה בעברית (RTL) המציג את קורס ההכנה ללידה, כולל:
- Hero section עם לוגו וקריאה לפעולה
- עמודי התווך של הגישה
- סילבוס מפורט של הקורס
- כפתור יצירת קשר בוואטסאפ

## טכנולוגיות

- HTML5 + CSS3 (Vanilla)
- פונט: [Heebo](https://fonts.google.com/specimen/Heebo) מ-Google Fonts
- עיצוב RTL מלא

## קבצים

| קובץ | תיאור |
|------|--------|
| `index.html` | דף הנחיתה הראשי |
| `logo.png` | לוגו |
| `logo-brand.png` | לוגו גרסת מותג |
| `logo-transparent.png` | לוגו על רקע שקוף |
| `hero-image.png` | תמונת hero |

## הרצה מקומית

פתחי את `index.html` ישירות בדפדפן — אין צורך בשרת.
